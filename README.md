# Stars

## 目录

### 文章
[weekly](https://github.com/ruanyf/weekly)：科技爱好者周刊，每周五发布。

[article](https://github.com/ICKelin/article)：读书笔记，博客文章。

[Halfrost-Field](https://github.com/halfrost/Halfrost-Field)：笔记文章。

### C++
[awesome-cpp](https://github.com/fffaraz/awesome-cpp)：C++框架，库。

[Wallpaper](https://github.com/NoisyWinds/Wallpaper)：把 h5 网页做成 Windows 动态壁纸(可hover交互)。

[WeChatRobot](https://github.com/TonyChen56/WeChatRobot)：微信HOOK、微信机器人 wxhook，数据库解密 微信公众号采集 微信公众号爬虫，企业微信HOOK。

[x64dbg](https://github.com/x64dbg/x64dbg)：Windows的开源用户模式调试器。优化逆向工程和恶意软件分析。

### C#
[N_m3u8DL-CLI](https://github.com/nilaoda/N_m3u8DL-CLI)：开源的命令行m3u8/HLS/dash下载器，支持普通AES-128-CBC解密，多线程，自定义请求头等。

[downkyi](https://github.com/leiurayer/downkyi)：哔哩哔哩网站视频下载工具，支持批量下载，支持8K、HDR、杜比视界，提供工具箱（音视频提取、去水印等）。

### Java
[ghidra](https://github.com/NationalSecurityAgency/ghidra)：软件逆向工程（SRE）框架。

### Go
[go-cqhttp](https://github.com/Mrs4s/go-cqhttp)：cqhttp的golang实现，轻量、原生跨平台。

[nps](https://github.com/ehang-io/nps)：一款轻量级、高性能、功能强大的内网穿透代理服务器。支持tcp、udp、socks5、http等几乎所有流量转发，可用来访问内网网站、本地支付接口调试、ssh访问、远程桌面，内网dns解析、内网socks5代理等等……，并带有功能强大的web管理端。

[go-captcha](https://github.com/wenlng/go-captcha)：行为验证码，实现了点击模式，滑动模式，拖放模式和旋转模式。

### Python
[ChatGLM-6B](https://github.com/THUDM/ChatGLM-6B)：开源双语对话语言模型。

[AutoGPT](https://github.com/Significant-Gravitas/AutoGPT)：AutoGPT的愿景是为每个人提供可访问的AI，用于使用和构建。我们的使命是提供工具，这样你就可以专注于重要的事情。

## JavaScript

[CefDetectorX](https://github.com/ShirasawaSama/CefDetectorX)：检测你电脑上有几个CEF。
